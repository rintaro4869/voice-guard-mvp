### 説明文

```
インターホンが鳴った瞬間、ワンタップで男性の声が応答。
「ひとり防犯ボイス」は、一人暮らしの方の宅配・来客対応をサポートする無料の防犯アプリです。

■ こんな不安はありませんか？
・宅配の対応で「一人暮らし」だと知られたくない
・インターホン越しに女性の声だけで応答するのが不安
・置き配を頼みたいけど、声をかけるのが怖い

■ 使い方はかんたん3ステップ
1. セリフと声のタイプを選んでおく
2. インターホンが鳴ったらアプリを開いて再生ボタンを押す
3. iPhoneのスピーカーをインターホンのマイクに近づける

■ 特長
・「はい」「玄関の前に置いといてください」など宅配対応に必要な5つのセリフを収録
・声のタイプは「丁寧」「素っ気ない」の2種類＋毎回ランダムに変わる「おまかせ」
・「3秒後に再生」機能で、スマホをインターホンに構えてから鳴らせる
・マナーモード中でも音が鳴るので、とっさの場面でも安心
・完全オフライン動作。アカウント登録・個人情報の入力は一切不要
・広告なし・完全無料

■ ご注意
本アプリは防犯の補助を目的としたものであり、安全を完全に保証するものではありません。身の危険を感じた場合は、ためらわず警察（110番）に連絡してください。
```

### 審査メモ（Notes欄）

```
このアプリはログイン不要で、すべての機能をそのままお試しいただけます。

【アプリの目的】
一人暮らしの方（特に女性）が宅配・来客対応をする際に、男性の声で応答することで「一人暮らしだと悟られない」ようにするための防犯補助アプリです。日本では宅配を装った犯罪への不安が社会的な課題となっており、本アプリはその対策ニーズに応えます。

【ネイティブアプリである理由】
・マナーモード（サイレントスイッチON）でも確実に音声を再生（AVAudioSessionの.playbackカテゴリ使用）。緊急時の確実な動作はWebでは実現できません
・完全オフライン動作（音声は端末内蔵）
・3秒カウントダウン再生で、スマホをインターホンに近づけてから再生可能

【音声について】
収録音声はすべて本アプリのために制作したオリジナル音源で、権利上の問題はありません。
```

## 提出手順（ユーザー作業）

1. **Xcodeでプロジェクトを開く**: `ios/VoiceGuard/VoiceGuard.xcodeproj`
2. **Archive**: 上部のデバイス選択を「Any iOS Device (arm64)」→ メニュー Product → Archive
   - 初回はAutomatic SigningがApp ID（com.voiceguard.hitoribouhan）と証明書を自動作成
3. **アップロード**: Organizerが開いたら Distribute App → App Store Connect → Upload
4. **App Store Connectでアプリ枠を作成**: https://appstoreconnect.apple.com
   - マイApp → ＋ → 新規App → プラットフォーム iOS / 名前・Bundle ID・SKU（voiceguard001など）を入力
5. 上記の「入力内容」をコピペで埋め、スクリーンショット3枚をアップロード
6. ビルドを選択 → 審査へ提出

## 審査リスクと対策

- **ガイドライン4.2（最小機能）が最大のリスク**。音声再生だけのアプリは「Webで十分」と判断されることがある
  - 対策: 審査メモに「マナーモードでも再生＝ネイティブでしか実現できない」を明記済み
  - リジェクトされた場合の追加機能候補: ホーム画面ウィジェット、Siriショートカット対応、セリフの追加収録
- 音声の権利確認: オリジナル音源である旨を審査メモに記載済み（要事実確認：音源の出どころが第三者素材の場合は記載を修正すること）
