# 置き配、便利だけど「盗まれないか不安」を解消する方法

一人暮らしの女性にとって、置き配はとても便利なサービスです。配達員と対面する必要がなく、不在時にも荷物を受け取れる。

でも「玄関前に放置された荷物が盗まれないか」という不安、ありませんか？

## 実際の盗難率は0.01%未満

Amazonの発表によると、置き配の盗難報告は全配送の0.01%未満。確率としてはかなり低いです。

とはいえ「自分が被害に遭わない保証はない」という心理的な不安は理解できます。日本の女性単独世帯は推計630万〜950万世帯。その多くが同じ不安を抱えています。

## 今日からできる6つの防止策

詳しくは記事にまとめましたが、ポイントだけ紹介します。

**1. 配達通知をオンにして即回収**
最も簡単で最も効果的。荷物が玄関前にある時間を最小限にすることが鉄則です。

**2. 置き場所を具体的に指定する**
「玄関前」だけだと目立つ場所に置かれがち。メーターボックス内や死角になる位置を指定しましょう。

**3. 簡易宅配ボックスを設置**
2,000〜5,000円で購入可能。ワイヤーロックで固定すればボックスごと盗まれるリスクも減ります。

**4. 在宅アピールをする**
荷物の盗難防止だけでなく、空き巣対策にもなります。

**5. 防犯カメラ（ダミーでも可）**
「監視されている」という心理的効果は絶大。

**6. 配達時間帯を在宅時に指定**
当たり前のようで、設定していない人が意外と多いです。

「ひとり防犯ボイス」を使えば、インターホン越しに男性の声で「玄関の前に置いといてください」と伝えられます。
