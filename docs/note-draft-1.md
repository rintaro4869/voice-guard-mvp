# 一人暮らしでインターホンが鳴ったとき、居留守を使っていませんか？

一人暮らしをしていると、突然のインターホンに「出たほうがいいのか、無視すべきか」と迷う場面がありますよね。

日本には一人暮らし世帯が約2,120万世帯あり、そのうち女性の単独世帯は推計630万〜950万世帯。つまり、この不安を感じている女性は何百万人もいるということです。

私の周りでも「基本は居留守」という女性が多いのですが、実は居留守にはリスクがあります。

## 居留守の最大のリスク

空き巣犯の多くが、犯行前にインターホンを押して「不在確認」をしています。

つまり、居留守で無反応だと「不在」と判断され、侵入のターゲットにされる可能性があるんです。

警察庁の統計では、住居侵入の約半数が玄関からの侵入。配達対応をきっかけに「この部屋は女性の一人暮らしだ」と知られることへの不安は、決して大げさではありません。

## 居留守より安全な方法がある

答えはシンプルで、**「不在」ではなく「在宅」を示すこと**です。

でも、自分の声（女性の声）で応答すると、一人暮らしの女性だとバレてしまう。

そこで作ったのが「ひとり防犯ボイス」です。

**男性の声でインターホンに応答できるWebアプリ**で、ワンタップで「はい」「玄関の前に置いといてください」などの短いフレーズを再生できます。

インストール不要・無料・登録不要で、スマホのブラウザからすぐに使えます。

## 既存アプリとの違い

似たようなアプリはいくつかありますが（「コエマモ」「男性応答アプリ 防犯」「男性の声素材集」など）、ひとり防犯ボイスは**配達対応に完全特化**しているのが特徴です。

- 必要なフレーズだけに絞っているから迷わない
- 声タイプを切り替えられるから、同じ配達員にもバレにくい
- ブラウザで動くからアプリのインストールが不要
