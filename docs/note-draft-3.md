# 一人暮らし女性の防犯対策、何から始めればいい？【7つの習慣】

一人暮らしの防犯対策って、「やったほうがいいと分かっていても、何から始めればいいか分からない」ものですよね。

日本の単独世帯は約2,120万世帯。そのうち女性の一人暮らしは推計630万〜950万世帯にのぼります。にもかかわらず、具体的な防犯対策をしている人はまだまだ少ないのが現状です。

でも安心してください。高い防犯グッズを買う必要はありません。**日常の習慣を少し変えるだけで、防犯力は大きく上がります**。

## 7つの防犯習慣（一部紹介）

**習慣1：帰宅時に後ろを確認してから鍵を開ける**
最も狙われやすい瞬間は「玄関の鍵を開けるとき」。背後を確認する → 素早く鍵を開ける → 入室 → すぐ施錠。この流れを習慣にするだけで安全性が大きく変わります。

**習慣2：インターホンに自分の声で出ない**
女性の声で応答すると「この部屋は女性の一人暮らし」と判断される可能性があります。モニターで確認するか、男性音声ツールを活用しましょう。

**習慣3：洗濯物から一人暮らしを悟らせない**
女性物だけが干してあると、外から丸わかりです。男性用のTシャツを一緒に干す（100均で買える）だけでも効果あり。

**習慣4：SNSで自宅が特定される情報を出さない**
ベランダからの景色、近所の店、「今から旅行！」の投稿... これらは全て自宅特定のヒントになります。

**残りの3つの習慣は記事で紹介しています**（カーテンの選び方、ゴミ出しの注意点、いざというときの連絡先リスト）。

## 防犯の基本は「犯罪者に選ばれない家にすること」

犯罪者は「入りやすそうな家」「住人が無防備な家」を選びます。つまり、防犯意識があることを外に示すだけで、犯罪のターゲットにされにくくなります。
