# VoiceGuard グローバル市場調査ベース戦略書

更新日: 2026-05-22

## 0. 結論

VoiceGuard は `男性音声アプリ` として売るべきではない。

勝ち筋は、`一人暮らし女性の生活防犯の最初の入口` になることです。  
つまり、無料の即時ツールで信頼を取り、その後に `比較ページ` `低単価デジタル商品` `高単価の安全対策リード` で収益化する。

優先順位は以下。

1. `無料ツールとして使われること`
2. `玄関・配達・帰宅不安の比較導線を育てること`
3. `物販以外の収益源を複線化すること`

## 1. この戦略の前提

- VoiceGuard の現在価値は `今すぐ使える安心` であり、継続課金される高度機能ではない
- 過去90日実績は `22クリック / 売上0件 / 成果報酬0円`
- 来訪意図は `買い物` ではなく `今すぐ不安を軽くしたい`
- したがって、`トップで売る` のではなく `不安を言語化してから売る` 設計が必要

## 2. 国内市場の示唆

### 2-1. すでに直接競合が複数ある

日本では、VoiceGuard と近い発想のアプリがすでに複数存在します。

- `１人暮らしの女性に便利な男性の声素材集` は、2025年4月・5月にもレビューが付き続けている
- `お守りサウンド` は `一人暮らし女性向け男性ボイスで防犯` を明示している
- `コエマモ` は `男性音声での代理応答` `フェイク通話` `防犯ブザー` `緊急通知` を束ね、App Store では `1か月300円 / 1年3,000円`

ここから言えるのは、`男性音声だけ` では差別化しにくく、課金上限も低いということです。

### 2-2. 問題自体は本物

一方で、不安の根は軽くありません。

- 警察庁は、宅配事業者を装った強盗等への対策として、`インターフォン等を通じた非対面受取` と `置き配等` の活用を周知している
- 警察庁の 2024 年データでは、ストーカー事案の相談等件数は `19,567件`、被害者の `86.4%` が女性
- 同データでは被害者の最多は `20歳代 35.1%`

つまり、`一人暮らし女性の自宅・来訪・帰宅不安` というテーマ自体には現実の需要がある。

## 3. 海外市場の示唆

### 3-1. 海外でお金が取れているのは「音声」ではなく「安全ワークフロー」

海外事例を見ると、収益化できているサービスは共通して `単発機能` ではなく `安全行動の一連の流れ` を売っています。

- `Noonlight`
  - 緊急通報、Safety Network、Timeline、Apple Watch、Uber/Lyft 連携
  - 公式サイトでは `3M+ people protected`
  - App Store では `Instant Access $4.99`、`Noonlight Premium $9.99`
- `Birdie+`
  - 女性向け防犯アラームにアプリ機能を接続
  - App Store では `Birdie+ monthly $4.99` / `annual $49.99`
  - `On-Call Rep` `Excuse-to-Leave Call` `Location Sharing` を提供
- `Parry`
  - 不快な場面から抜ける `フェイク着信` と `位置共有`
  - App Store では `Monthly $1.99`、`Yearly $19.99`、`Lifetime $4.99`
- `Portixa`
  - ブラウザベースで `静かに共有する` ことに特化
  - `15日無料` の後、`$5.99/月` または `$49.99/年`
- `Cove`
  - 位置バレやデート時のデジタル安全に特化
  - `Safety score` は無料、保護機能は `$9.99/月` または `$79.99/年`

### 3-2. 重要な学び

海外の勝ち筋から得られる示唆は5つあります。

1. `ただの便利機能` は安い  
   Parry の価格帯は、VoiceGuard 単体課金の上限がかなり低いことを示す。
2. `日常の安全習慣` まで入ると月額化できる  
   Noonlight や Cove は、単機能ではなく生活文脈を取っている。
3. `ハード + サービス` は成立する  
   Birdie+ はアラーム単体ではなく、アプリと継続機能で収益化している。
4. `静かな安全` にも市場がある  
   Portixa は、監視や大げさなアラートではなく `気づかれず備える` を売っている。
5. `B2B2C` に拡張できる  
   Tinder は Noonlight と連携しており、個人向け安全機能はプラットフォーム連携にもなりうる。

## 4. VoiceGuard が取るべきポジション

VoiceGuard の強みは、海外勢のような常時監視や緊急通報ではありません。

強みは次です。

- 登録不要
- インストール不要
- 1タップで使える
- 日本の `インターホン` `訪問販売` `宅配対応` に文脈が合う

したがって、最適なポジションはこうです。

`日本の一人暮らし女性向け、玄関・配達・帰宅不安の最短入口`

ここから外れると弱くなります。

- `総合防犯アプリ` を目指す: 機能競争で負けやすい
- `有料の男性音声アプリ` を目指す: 単価が小さすぎる
- `一般ニュース型の防犯メディア` を目指す: 個人開発では広すぎる

## 5. 収益モデル評価

### A. 比較ページ + 物販アフィリエイト

評価: `最優先`

理由:

- すでに実装済みの延長で動ける
- 売る場所を `recommend-*` に限定できる
- 来訪意図に近い `玄関対策` `持ち歩き対策` に絞れる

対象カテゴリ:

- 防犯ブザー
- 賃貸向け補助錠
- ドアスコープカバー
- 窓用防犯センサー
- スマートドアホン
- 宅配ボックス / 置き配バッグ

注意:

- `トップで売らない`
- `子ども向け` が強すぎる商材は後回し
- 商品名より `不安解決シナリオ` を前面に出す

### B. 低単価デジタル商品

評価: `かなり有望`

理由:

- 単価は低くても粗利が高い
- 個人開発と相性が良い
- 物販のように在庫切れで崩れにくい

売るべきもの:

- `引っ越し初日の防犯チェックリスト`
- `0円 / 3,000円 / 15,000円 防犯プラン`
- `訪問販売・配達対応の断り文句テンプレ`
- `一人暮らし女性の玄関防犯ミニガイド`

価格帯の目安:

- 無料: リスト取り用
- 300〜980円: impulse buy
- 1,480〜2,980円: 情報密度が十分ある場合のみ

避けるべきもの:

- 情報量の薄い有料PDF
- 汎用的すぎる `防犯まとめ`

### C. 高単価リード獲得 / スポンサー

評価: `中長期で有望`

理由:

- 住まいの安全領域は、月額数千円の商材が実在する
- セコムは集合住宅向けで `月額3,520円〜`
- ALSOK は集合住宅向けで `月額3,300円〜`、セルフセキュリティで `月額990円〜`

VoiceGuard 単体では売れなくても、

- 一人暮らし女性向けの防犯ハブ
- 引っ越し直後の安全対策
- 賃貸でできる玄関強化

まで文脈を広げれば、スポンサーや送客の余地がある。

現時点の現実路線:

- まずは広告代理店的に考えない
- 比較記事と CTA 実績を作る
- その後に `女性向け賃貸` `防犯設備` `スマートドアホン` 領域へ営業する

### D. LINE / メール / オウンド読者化

評価: `必要`

理由:

- 今のトラフィック規模では、一度の訪問で収益化し切るのは難しい
- `春の引っ越し` `夏の帰宅不安` `年末防犯` のような再接触が必要

ただし、やり方は軽くする。

- トップには置かない
- 比較ページやガイドページでのみ案内する
- まずは `無料チェックリスト` と交換で十分

### E. 有料アプリ / サブスク

評価: `やらない`

理由:

- 国内でも類似アプリが複数ある
- 海外でも単機能アプリの価格は低い
- VoiceGuard の価値は `即時性` であり、継続課金の説得力が弱い

結論:

- 買い切りアプリ: 検証対象にも優先しない
- サブスク化: 現状では不採用

## 6. 事業モデルの再定義

VoiceGuard をこう再定義する。

`無料の防犯ユーティリティ` ではなく  
`一人暮らし女性の生活防犯メディア兼ツール`

収益構造はこうする。

1. `ツール`
   - 音声再生で信頼獲得
2. `導線`
   - safety-plan で不安を分岐
3. `比較`
   - recommend ページで選択を助ける
4. `収益`
   - 物販
   - デジタル商品
   - リード獲得
   - 将来的なスポンサー

## 7. コンテンツの勝ち筋

### 第1テーマ: 玄関・配達対応

最重要。

理由:

- プロダクトと最も自然につながる
- 警察庁の `非対面受取` 文脈とも一致する
- 商材の幅がある

作るべきページ例:

- `女性一人暮らしでインターホンが怖いときの対策`
- `置き配にしたいけど不安な人向けの対策`
- `訪問販売を玄関で断るテンプレ集`
- `賃貸でもできる玄関防犯グッズ比較`

### 第2テーマ: 夜道・持ち歩き

次点。

理由:

- 防犯ブザーやライトに接続しやすい
- ただし VoiceGuard 本体との接続は玄関より弱い

作るべきページ例:

- `防犯ブザーは大人の女性にも必要か`
- `通勤帰宅で持っておくべき防犯アイテム`
- `夜道が不安なときの無料対策と有料対策`

### 第3テーマ: 引っ越し直後の安全対策

収益化しやすい。

理由:

- ニーズがはっきりしている
- 物販・PDF・高単価送客が全部乗る

作るべきページ例:

- `女性一人暮らしの部屋選び防犯チェック`
- `入居初日にやるべき防犯5つ`
- `賃貸でできる窓・玄関の強化`

### 第4テーマ: デート・位置バレ・静かな安全

拡張候補。

理由:

- 海外では強いテーマ
- Portixa や Cove が取っている需要に近い

ただし、今すぐ主軸にはしない。  
VoiceGuard との距離が少しあるため、玄関・帰宅の基盤ができてから広げる。

## 8. 90日実行計画

### 0〜30日

目的: `何が刺さるかを見極める`

やること:

- 既存ファネルの計測確認
- `recommend-intercom` と `recommend-buzzer` の比較
- 玄関テーマの記事を3本追加
- デジタル商品の無料版を1つ作る

判定:

- `トップ -> safety-plan`
- `safety-plan -> recommend-*`
- `recommend-* -> aff_click`

### 31〜60日

目的: `収益源を二本目まで作る`

やること:

- `引っ越し防犯チェックリスト` を公開
- メール or LINE の軽い受け皿を作る
- 玄関テーマの比較ページを追加
- 置き配 / 宅配ボックス関連ページを追加

判定:

- 比較ページ別の CTR
- デジタル商品のダウンロード率 / 購入率

### 61〜90日

目的: `勝ち筋へ集中する`

やること:

- 勝ったテーマに記事と CTA を寄せる
- 必要ならスポンサー向け資料を作る
- 住まい系 / 防犯系への協業打診を開始する

判定:

- 売上の有無より、`どのテーマで商用クリックが出るか`
- そのテーマに再現性があるか

## 9. 絶対にやらないこと

- トップページを広告っぽくする
- VoiceGuard 本体を有料アプリとして売る
- いきなり総合防犯サービス化する
- 商品リンクをベタ貼りする
- SEO流入前提で広すぎる一般防犯メディアを作る

## 10. 判断基準

次のどれかが立てば、事業として前進していると見ていい。

1. `玄関対策` でクリックが集まる
2. `夜道対策` でクリックが集まる
3. `引っ越し防犯` の資料請求やDLが取れる
4. `女性一人暮らしの安全対策` の文脈で協業先が現れる

逆に、次なら縮小判断です。

- 比較ページ送客が増えない
- 外部クリックが出ない
- 無料ツールとしては使われるが、周辺テーマに広がらない

その場合は、VoiceGuard を `便利な小さな資産` として維持し、過剰投資しない。

## 11. いま取るべき最重要戦略

VoiceGuard の最重要戦略はこれです。

`無料ツールで信用を得て、玄関・配達・引っ越し防犯の意思決定メディアになる`

この形なら、

- 既存プロダクトとつながる
- 個人開発でも運用できる
- 物販だけに依存しない
- 海外事例の学びも取り込める
