# VoiceGuard 実行ロードマップ

更新日: 2026-05-23

## 目的

- `無料ツール -> 比較 -> 収益` の流れを実装レベルで前に進める
- 90日以内に `玄関対策` `夜道対策` `引っ越し防犯` のどれが勝ち筋か判定する
- 物販だけに依存しない収益源を2本以上作る

## 最重要KPI

1. `safety-plan -> recommend-*` 遷移数
2. `recommend-* -> aff_click` クリック率
3. `無料チェックリスト` ページ流入数
4. `無料チェックリスト` ページから比較ページへの遷移率
5. `玄関系` と `夜道系` のどちらにクリックが寄るか

2. その資産から比較ページへ流す
3. 比較で勝つテーマを見つける
4. 勝ったテーマだけ深掘る
