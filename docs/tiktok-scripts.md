# TikTok動画企画 & 台本集

## バズるための基本ルール
- **最初の1秒で止める**（スクロールを止めさせるフック）
- **15〜30秒が最適**（視聴完了率がアルゴリズムの最重要指標）
- **テキストを画面に出す**（音声オフで見る人が多い）
- **最後まで見たくなる構成**（オチを最後に持ってくる）
- **投稿時間**: 平日20:00〜22:00 or 昼12:00〜13:00

## 動画1：「一人暮らしあるある × 解決」型（最もバズりやすい）

### コンセプト
インターホンが鳴った瞬間の恐怖 → 男性音声で解決

### 台本（15秒）
```
[画面テキスト] 一人暮らし女子あるある

[0秒] インターホンの「ピンポーン」の音
[1秒] テキスト表示：「え、誰…？」
       女性がスマホを持って固まる演技

[3秒] テキスト表示：「出たら女の一人暮らしってバレる」
       不安な表情

[5秒] テキスト表示：「でも居留守は空き巣の不在確認かも」
       さらに不安

[7秒] テキスト表示：「そこで…」

[8秒] スマホ画面を映す（アプリを開く）
       テキスト表示：「ワンタップで男性の声」

[10秒] 再生ボタンをタップ
        男性の声で「はい」が流れる

[12秒] テキスト表示：「無料・登録不要・ブラウザだけ」
        安心した表情

[14秒] テキスト表示：「プロフのリンクから使えます」
```

### BGM
- 緊張感のある音 → 解決で明るい音に切り替え
- または「あるある」系でよく使われるトレンド音源

## 動画2：「実際にやってみた」型（信頼性重視）

### コンセプト
実際のインターホン越しにどう聞こえるかデモ

### 台本（20秒）
```
[画面テキスト] 男性の声でインターホンに応答してみた

[0秒] テキスト：「一人暮らしの配達対応が怖い人へ」

[2秒] インターホンのある玄関を映す
       テキスト：「実際にどう聞こえるか試してみた」

[4秒] インターホンが鳴る
       スマホを取り出す

[6秒] アプリ画面をアップで映す
       「玄関の前に置いといてください」をタップ

[8秒] 男性の声が再生される
       テキスト：「インターホン越しだとこう聞こえる↓」

[10秒] ドアの外側からの音を録音したものを再生
        テキスト：「ちゃんと男性が住んでる感出てる」

[14秒] テキスト：「声タイプも変えられる」
        別の声タイプで再生

[17秒] テキスト：「無料です。プロフから」
```

[13秒] 7. 緊急連絡先（#9110）を登録しておく

[15秒] テキスト：「2番の解決策がこれ」
        アプリ画面を映してデモ
